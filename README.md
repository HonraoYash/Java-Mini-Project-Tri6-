PIGGY BANK Management System

The Piggy Bank Management System is a web-based application used for 
the financial transactions replicating a bank-website . 
This Project is based on Client-Server model . Eclipse IDE for Enterprise Java and Web Developers was used to edit the code . Apache Tomcat Server was implemented along with  JSP, Servlets and Java Database Connectivity (JDBC) . MySQL database was used for storage .
